# Daan's Persoonlijke Hulp (Personal Assistant)

A comprehensive personal productivity and life management system built as a "life OS" with AI-powered natural language interaction.

## 🌈 Wat is dit voor een app? (Kindertaal)

Hoi! Dit is de app van Daan. Het is een soort super-hulpje dat Daan helpt om alles goed te onthouden. Zie het maar als een digitale rugzak waar je alles in kunt stoppen!

- **📝 Briefjes en lijstjes**: Heb je een goed idee? Schrijf het op! Moet je je kamer opruimen? Zet het op je lijstje!
- **💰 Sparen en centjes**: De app houdt bij hoeveel geld Daan heeft voor snoepjes of speelgoed.
- **🤝 Vriendjes en familie**: Alle namen van vriendjes staan handig bij elkaar.
- **🌟 Goede gewoontes**: De app helpt Daan om elke dag goed zijn tanden te poetsen of een appel te eten.
- **🤖 Praten met de computer**: Er zit een slimme robot in waar Daan vragen aan kan stellen!

## Tech Stack
- **Frontend**: Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS
- **Database**: Neon Serverless PostgreSQL
- **AI**: OpenAI API (GPT-4o-mini) for natural language processing
- **Rich Text**: Tiptap editor for notes
- **Icons**: Lucide React

## Features
- **Dashboard**: Overview of everything happening today.
- **Chat**: AI assistant for natural language commands.
- **Todos**: Task management with priorities and due dates.
- **Finance**: Tracking income and expenses.
- **Habits**: Daily and weekly habit tracking.
- **Journal**: Personal diary and mood tracking.
- **And much more**: Contacts, Ideas, Projects, Worklogs, etc.
